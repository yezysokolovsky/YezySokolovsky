# ML Classification HW4

This project compares multiple classification models on the Breast Cancer dataset.

## Models
Gradient Boosting, CatBoost, AdaBoost, Extra Trees, QDA, LightGBM, KNN, Decision Tree, XGBoost, SVM

## Metrics
Accuracy, Precision, Recall, F1-score, ROC-AUC

## How to run
pip install -r requirements.txt
jupyter notebook
