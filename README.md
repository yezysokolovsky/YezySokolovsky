
# Final Project - Smart Club Energy Forecasting

## Project topic
Forecasting energy consumption of a sports club and detecting anomalies.

## Files
- `data/smart_club_energy_dataset.csv` - dataset
- `notebooks/final_project_smart_club_energy.ipynb` - full notebook with EDA, preprocessing, ML models and anomaly detection
- `requirements.txt` - dependencies

## Task coverage
This project includes:
- new dataset;
- feature description;
- EDA;
- preprocessing;
- time series analysis with at least 5 ML models;
- anomaly detection;
- forecast error plots, model metric plots and feature importance.

## Models
- Linear Regression
- Ridge
- Lasso
- Random Forest Regressor
- Gradient Boosting Regressor
- SVR

## Anomaly detection
- Isolation Forest

## How to run
```bash
pip install -r requirements.txt
jupyter notebook
```
